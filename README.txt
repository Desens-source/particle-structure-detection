# Particle Structure Detection

This repository contains the Python code for detecting and verifying particle structure detachments in high-speed video, as described in the manuscript for MethodsX.

## Contents

- `01_Particle_structure_detachment_detection.py` – Modul 1: Detektion von Partikelstrukturen  
- `02_Checking_the_detected_particle_structure_detachments.py` – Modul 2: Verifikation der detektierten Ereignisse 
- `requirements.txt` – benötigte Python-Pakete
